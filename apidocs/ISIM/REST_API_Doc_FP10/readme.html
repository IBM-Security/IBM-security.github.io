Readme file for IBM Security Identity Manager REST APIs

The document contains information about the IBM Security Identity Manager REST APIs.

Overview

IBM Security Identity Manager provides a set of APIs that you can use to administer the tasks outside of the IBM Security Identity Manager user interface.

The REST APIs are segregated into a set of functional components of IBM Security Identity Manager that are listed in the following section.
  • Person management
  • System User management
  • Password management
  • Access management
  • Activity management
  • Delegation management
  • Generic Search APIs

How to use the REST API documentation

  1. Open the index.html file. See the "Introduction" section and list of APIs.
  2. Expand items in the left pane to find documentation for the specific task.
You can find more information about REST APIs in the IBM Security Identity Manager 6.0.0.10 Knowledge Center at http://www.ibm.com/support/knowledgecenter/SSRMWJ_6.0.0.10/com.ibm.isim.doc_6.0.0.10/reference/cpt/cpt_ic_isim_rest_api_overview.htm.
